# apollo-continuous-integration
Continuous Integration with Apollo Server

![GitHub Actions](https://github.com/nih0n/apollo-continuous-integration/workflows/Release/badge.svg)

## Installation
```bash
yarn install
yarn build
```

## Running
```bash
yarn start
```

#### Query
```graphql
{
  hello
}
```

#### Response
```graphql
Hello, World!
```
